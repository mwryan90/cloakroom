# cloakroom-ui

Localhost admin UI for [cloakroom](https://github.com/mwryan90/cloakroom), a masking proxy for MCP servers: review columns, toggle sensitivity, assign or rename tokens, exclude placeholders, and browse mappings. Binds to `127.0.0.1` only.

Most users launch it through the [`cloakroom`](https://www.npmjs.com/package/cloakroom) CLI package:

```bash
npx cloakroom ui
```
